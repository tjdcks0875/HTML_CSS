1. 드롭다운 목록을 만들어주는 <select>
    - 사용자가 직접 내용을 입력하지 않고 여러 옵션중 선택할 때 사용
    - <select>로 드롭다운 목록의 위치를 지정
    - <option>로 여러가지 선택지를 추가한다.
    - value 속성 : 서버단 프로그램으로 전송될 값 지정
    - selected : 초기 화면에 표출될 때 선택표시될 옵션
2. 옵션을 그룹화해주는 <optgroup>
    - 비슷한 내용의 <option>을 묶어서 표출할 때 사용
    - label 속성 : 그룹의 이름을 지어줌
3. 여러 줄 입력 텍스트 영역을 만들어주는 <textarea>
    - 한 줄 이상의 문장을 입력받을 때 사용
    - 가입인사를 입력하거나 회원약관을 표시할 때 주로 사용
4. submit 버튼에 이미지 넣기
    - input type="image"로 지정하면 이미지가 들어간 submit 버튼이 생성된다
    - src 속성으로 이미지의 경로지정
5. 버튼을 생성해주는 <button>
    - <button>를 사용해도 submit, reset과 동일한 버튼 추가 가능
    - <button> 생성 시 기본 type은 submit
    - submit : 서버단 프로그램으로 전송
               <form>의 action을 동작
    - reset : 폼 초기화
    - button : 일반 버튼, 아무 기능 없음
               onclick 이벤트를 등록하여 기능을 추가
6. 시맨틱 태그
    - HTML5 지원된 태그들
    - HTML 문서의 구조를 잡는 데 도움을 주는 태그들
    - <header> : HTML 문서의 상부(헤더부분) 요소들을 감쌈
                 로고, 네브바(메뉴바), 검색창 등이 포함됨
    - <nav> : 네비게이션 메뉴를 감쌈
              header 태그안에 포함되는 경우가 많다.
    - <article> : 기사부분 태그
    - <section> : 테마를 가진 그룹만드는 태그
    - <main> : 페이지의 메인콘텐츠가 되는 부분
               한 문서당 하나의 main 태그만 존재해야됨
    - <aside> : 추가정보를 표시하는 태그
                 우측이나 좌측 양사이드에 위치
    - <footer> : 페이지의 하단부를 만드는 태그
                 일반적으로 이용약관, 저작권, contact등을 표시
    - <div> : 아무의미 없는 태그
              가장 많이 사용되는 태그
              요소들을 그룹으로 묶어서 스타일 적용할 때 사용
7. CSS란
    - Cascading Style Sheet의 약자로 HTML문서에 디자인을 입혀주는 역할
    - HTML 태그들이 문서 뼈대를 담당, CSS는 디자인 요소를 담당
    - 웹 문서의 내용과 상관없이 디자인만 변경가능
    - 다양한 기기에 맞춰 반응형 웹사이트를 제작할 수 있다(미디어 쿼리 사용)
8. Style Sheet의 종류
    - 외부 스타일 시트 : 별도의 css파일을 만들어서 스타일 지정
                        html문서에서 head부분에 
                        link태그를 이용해서 불러옴
                        <link rel="stylesheet" href="style.css">
    - 내부 스타일 시트 : HTML 문서 head부분에 <style> 태그를 이용해서
                        스타일 지정
    - 인라인 스타일 시트 : 태그안에 직접 style 속성을 이용해서 스타일 지정
9. 스타일 시트 적용 순서
    - 인라인 스타일 시트 > 내부 스타일 시트 > 외부 스타일 시트
10. 스타일 지정 형식
    - 선택자 {
        속성 : 속성값;
    }
    - 선택자 : 스타일이 적용될 대상(태그, 클래스, 아이디 등)
    - 속성 : 어떤 스타일 변경할 지 지정
    - 속성 값 : 변경할 스타일을 어떻게 변경할 지 지정
    - 속성을 여러개 지정할 때는 ;을 붙여서 지정
11. 스타일 시트에 주석
    - /**/를 이용해서 주석을 달아줌
    - 한 줄 또는 여러줄로 주석 입력 가능
12. 선택자
    - 전체 선택자 : 모든 요소에 스타일을 적용할 때
                   * {
                       속성: 속성값;
                   }
                   가장 많은 요소들에 스타일 적용 가능
    - 태그 선택자 : 특정 태그에만 스타일 적용할 때
                    태그명 {
                        속성: 속성값;
                    }
                    전체 선택자 다음으로 가장 많은 요소에 스타일 적용 가능
                    ※ 태그와 요소의 차이점
                    - 태그는 문서의 구조를 잡을 때 사용하는 문법
                    - 요소는 태그안의 내용
                    - 스타일을 적용하는 것은 태그에 적용하는 게 아니고
                      태그안의 내용 즉, 요소에 스타일을 적용하는 것
    - 클래스 선택자 : 같은 태그에 다른 스타일을 적용하고 싶을 때
                     .클래스명 {
                         속성: 속성값;
                     }
                     만약 다른태그 같은 클래스일 경우
                     태그명.클래스명 {
                         속성: 속성값;
                     }
                     클래스 선택자는 가장 많이 사용됨
    - id 선택자 : 같은 태그에 다른 스타일을 적용하고 싶을 때
                  #id {
                      속성: 속성값;
                  }
                  id 속성은 한 문서안에서 유일(하나만 존재)
                  오로지 하나의 요소에만 스타일 적용할 때 id선택자 사용
    - 그룹 선택자 : 둘 이상 요소에 같은 스타일을 적용하고 싶을 때
                   이름1, 이름2, 이름3,... {
                       속성: 속성값;
                   }
                   ,로 구분해서 선택자 나열
                   이름에는 태그명, 클래스명, id명 등 다양한 종류 사용 가능
13. 선택자의 적용 순서
    - id선택자 > 클래스선택자 > 태그선택자
14. 스타일의 상속
    - 웹 문서에서 사용하는 태그들에는 포함관계가 존재
    - 포함하고 있는 태그를 부모요소,
      포함된 태그를 자식요소
    - 자식요소에 별도의 스타일 적용하지 않으면
      부모요소의 스타일 자식요소로 전달됨, 이것을 스타일의 상속이라고 함
15. 상속을 이용한 선택자
    - 자식 선택자 : >를 이용해서 부모요소에 포함돼있는 자식요소 선택
                   부모선택자>자식선택자 {
                       속성: 속성값;
                   }
    - 자손 선택자 : 공백을 이용해서 부모요소 하위에 있는 모든 요소를 선택
                   부모선택자 자손선택자 {
                       속성: 속성값;
                   }
16. 글꼴관련 스타일
    - font-familyl 속성 : 웹 문서에서 사용할 글꼴 지정
                          font-family: 글꼴1, 글꼴2, 글꼴3,....;
                          글꼴1이 존재하면 글꼴1로
                          글꼴1이 없으면 글꼴2로
                          글꼴2도 없으면 글꼴3
                          주로 body에 적용하여 웹 문서에 통일성을 부여
    - 웹 폰트 사용 : @import url 웹 폰트를 다운로드 하여 사용
    - font-size 속성 : 폰트 크기를 지정
    - 폰트 크기 지정방식
      절대 크기 : xx-small|x-small|small|medium|larg|x-large|xx-large
                 브라우저에서 지정한 폰트 크기
      상대 크기 : larger|smaller
                 부모요소의 폰트크기를 기준으로 크기 조절
      수치 : px, em, ex, pt 단위별로 수치 직접 지정
      백분율 : 부모요소의 폰트크기를 기준으로 %단위로 크기 조절
    - font-weight : 글자 굵기 조절
    - 폰트 굵기 지정 방식
      normal : 일반적인 굵기, 굵기 지정 안할 시 기본 값
      bold|lighter|bolder : 굵게|더 가늘게|더 굵게
      100~900 : 400이 normal. 작아질수록 가늘게 커질수록 굵게
    - line-height : 줄 간격 조절. 윗 문장과 아래 문장 사이의 길이 조절
                    1.5 ~ 1.9 사이로 지정하는 것을 권장함
    - color 속성 : 글자 색 지정하기
    - 색상 지정 방식
      16진수 값 지정 : 0 ~ 9, a ~ f를 이용하여 6자리 16진수로 지정
                      #ffffff => 하얀색
                      앞 두자리가 빨간색, 중간 두자리 초록색,
                      마지막 두자리는 파란색
                      #ffaabb => #fab
                      두 자리씩 중복 될때는 세자리로 표현 가능
      rgb 값 지정 : rgb(0 ~ 255, 0 ~ 255, 0 ~ 255)
                    첫 자리 빨간색, 두 번째 자리 초록색, 
                    세번째 자리 파란색
      rgba 값 지정 : rgb 지정방식에서 투명도 추가
                     rgba(0 ~ 255, 0 ~ 255, 0 ~ 255, 0 ~ 1)
                     0이 완전 투명, 1이 완전 불투명
      색상 이름 지정 : red, blue, black,... 등과 같이 이름으로 지정
    - text-align 속성 : 텍스트 정렬하기
    - 정렬 방식
      left : 왼쪽 정렬
      right : 오른쪽 정렬
      center : 가운데 정렬
      justify : 양쪽 정렬
    - list-style-type 속성 : 목록의 불릿 모양 지정하기
    - ul의 불릿 모양
      disc(●) : 채워진 원. 기본 값
      circle(○) : 비워진 원
      square(■) : 채워진 사각형
      none : 불릿 표시 안함
    - list-style-image 속성 : 불릿 대체 이미지 넣기
      none : 지정된 불릿 표시
      url(이미지경로) : 이미지로 불릿 표시
